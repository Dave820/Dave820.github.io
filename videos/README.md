# Videos Directory

This directory should contain the videos for each stage of the V-lifecycle diagram.

## Required Video Files

Place your MP4 video files and thumbnail images here with the following names:

### Video Files (MP4 format recommended):
- `requirements.mp4` - Requirements Analysis stage
- `design.mp4` - System Design stage
- `architecture.mp4` - Architecture Design stage
- `module.mp4` - Module Design stage
- `implementation.mp4` - Implementation stage
- `module-test.mp4` - Module Testing stage
- `integration.mp4` - Integration Testing stage
- `system-test.mp4` - System Testing stage
- `acceptance.mp4` - Acceptance Testing stage

### Thumbnail Images (JPG format recommended):
- `requirements-thumb.jpg` - Thumbnail for Requirements Analysis
- `design-thumb.jpg` - Thumbnail for System Design
- `architecture-thumb.jpg` - Thumbnail for Architecture Design
- `module-thumb.jpg` - Thumbnail for Module Design
- `implementation-thumb.jpg` - Thumbnail for Implementation
- `module-test-thumb.jpg` - Thumbnail for Module Testing
- `integration-thumb.jpg` - Thumbnail for Integration Testing
- `system-test-thumb.jpg` - Thumbnail for System Testing
- `acceptance-thumb.jpg` - Thumbnail for Acceptance Testing

## Notes

- Videos will show as live thumbnails when hovering over them
- Clicking a thumbnail expands the video to fullscreen
- Click anywhere or press ESC to return to the diagram
- Recommended video resolution: 1920x1080 or 1280x720
- Thumbnail resolution: 280x157 (16:9 aspect ratio)
- Keep video file sizes reasonable for web delivery (< 10MB per video recommended)
